# Pizzeria Order

Simple console application in Java using builder and factory design patterns.

## Technology Stack

- Java 8
- Maven

## App Run

Inside the root directory:

On Unix and Unix-like systems

```
cd src/main/java
javac com/example/pizzeria/*.java
java com.example.pizzeria.Main
```

On Windows systems

```
cd src\main\java
javac com/example/pizzeria/*.java
java com.example.pizzeria.Main
```
